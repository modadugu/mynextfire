# Testing Locally

To test your site locally before pushing to GitHub:

1. Install dependencies:
   ```bash
   bundle install
   ```

2. Run Jekyll server:
   ```bash
   bundle exec jekyll serve
   ```

3. Open in browser:
   ```
   http://localhost:4000
   ```

4. Access posts at:
   - Homepage: http://localhost:4000
   - Posts: http://localhost:4000/uncategorized/2025/03/02/pitch-perfect-crafting-your-business-pitch-with-ai.html
